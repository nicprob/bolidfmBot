# bolidfmBot

Telegram bot for Radio Bolid.

## Setup

1. Install dependencies:

   ```powershell
   pip install -r requirements.txt
   ```

2. Create a local `.env` file or set the environment variable directly:

   ```powershell
   $env:BOT_TOKEN="your-telegram-bot-token"
   ```

3. Run the bot:

   ```powershell
   python bot.py
   ```

Never commit real bot tokens, API keys, or `.env` files.
